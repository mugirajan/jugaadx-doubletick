## Import Customers

To import customers to the broadcast list, follow these steps:

* Click on the three dots icon <img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FPma9D7MbQ9443dPjvESl%2Fimage.png?alt=media&#x26;token=b2567dde-fd69-4c69-9ee7-c49bd5bbc511" alt="" data-size="line">
* Select "**Download sample excel format**"&#x20;

<div align="left"><figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2Fndx7oD4BOUhdTubYGi8B%2Fimage.png?alt=media&#x26;token=672bb100-360c-40a1-b8ae-dabc27f65528" alt=""><figcaption></figcaption></figure></div>

* A sample Excel file will be downloaded automatically with the following columns:

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FfRtb856ymjdtlUXoJaEu%2Fimage.png?alt=media&#x26;token=2db38181-b93c-428d-9331-c3706e09c183" alt=""><figcaption></figcaption></figure>

**Name:**

* Enter contact name
* ***Name is Non-Mandatory***

**Phone Number**:

* Enter the customer's contact number along with a "+" followed by the country code, and then the phone number.&#x20;
* For example, if the customer is from India, the number format should be:

  +91xxxxxxxxxx
* Here, "91" is the country code for India, followed by the customer's phone number
* ***Phone Number is Mandatory***

**Channel Name**:

* Enter the name of your broadcast list in the channel name field
* The name of the broadcast list entered here will determine where the customers are imported, ensuring they are added to the specified list.
* However, if there's no existing channel and you want to create a new one with the customers you're importing, simply type the desired channel name in the provided field
* After the import process, a new channel will be created with the specified name, containing the imported customers
* ***Channel Name is Mandatory***

**Agent Phone:**

* In the "Agent Phone" field, you should input the phone number of the agent whom you wish to assign the specific customer conversation to
* ***Agent Phone is Non-Mandatory***&#x20;

**Custom field 1:**

* In the custom field one, you can include any customer attribute you've created.&#x20;
* If you have multiple customer attributes, you can add multiple columns to include them.
* &#x20;For instance, if you want to add the customer attribute "**Birthdate,**" you can rename the custom field one to "**Birthdate**" and then enter the details.&#x20;
* Similarly, if you want to add another customer attribute, such as "**Location**," you can add another column and enter the location details.
* ***Custom field is Non-Mandatory***

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FIhyp0J5r2lRzqDuUY5sl%2Fimage.png?alt=media&#x26;token=b60c62af-baeb-40ae-9f1f-37a3045e6d44" alt=""><figcaption></figcaption></figure>

*<mark style="color:green;">**Note that you have the option to delete any non-mandatory fields if you don't need them.**</mark>*

* After entering all the details, save the file.&#x20;
* Then, return to DoubleTick platform &#x20;
* Click on the three dots again and select the "**Import contacts from Excel**" option.&#x20;

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2Ftjma4UilGWoK8toO7oYO%2Fimage.png?alt=media&#x26;token=cba019c6-08a1-499d-8716-324df80ccb5b" alt=""><figcaption></figcaption></figure>

* A pop-up window for importing customers will appear.&#x20;
* Click on the "**Browse**" button and select the file you want to import from your device.&#x20;

***This will allow you to navigate to the location where you saved the Excel file containing the customer information***&#x20;

* When you click on the import option, a dialog box "**Import Customer - Map Columns"** for importing customers will appear
* It consists of two main sections: "**Fields**" and "**Excel Fields**."

1. **Fields:** This section is further divided into two sub-sections:
   * **Static Fields:** These are the fields that are built into the system
   * **Custom Fields:** These are customer custom attributes created by the user
2. **Excel Fields:** This section displays the column names from the Excel file you selected for import

In this dialog box, you'll need to map the fields from the Excel file to the corresponding fields in the system. This ensures that the data is correctly imported and aligned with the appropriate fields in your system.

<div align="left"><figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FJmPyaJ0hUs9vySAblhj2%2Fimage.png?alt=media&#x26;token=90aad086-4539-470f-a5fc-5798f1d62338" alt="" width="331"><figcaption></figcaption></figure></div>

* To map the fields correctly,  click on the field column in the dialog box, then select the corresponding Excel field column name from the dropdown menu.&#x20;
* Repeat this process for each field until all fields are correctly mapped.

<div align="left"><figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FcUazelseUKLKS2KZum8i%2Fimage.png?alt=media&#x26;token=b3ea9339-637a-4630-b15f-e952246bb208" alt="" width="327"><figcaption></figcaption></figure></div>

* A confirmation pop-up will appear. Click on the "<mark style="color:green;">**Upload**</mark>" button

<div align="left"><figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FW3Ipvvk2lILsak9hwATn%2Fimage.png?alt=media&#x26;token=e81f8a30-764e-4ead-bf51-f81c1e240454" alt="" width="331"><figcaption></figcaption></figure></div>

* Once you've ensured that all fields are mapped correctly, click on the "<mark style="color:green;">**Import**</mark>" button to initiate the import process
* This will transfer the data from the Excel file into the system according to the specified mappings

***

Once the file is successfully imported, you will receive a confirmation notification via email.&#x20;

This notification is particularly helpful when dealing with large amounts of imported data, which may take a little more time to process.&#x20;

<div align="left"><figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FZsbWSRMHcDh1JsfDuAOi%2Fimage.png?alt=media&#x26;token=a407995d-2d7b-4916-9b98-5635fb3babcc" alt="" width="316"><figcaption></figcaption></figure></div>

It ensures that you are informed when the import process is completed and allows you to keep track of the progress.
