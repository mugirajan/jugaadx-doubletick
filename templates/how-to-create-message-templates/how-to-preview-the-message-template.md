# How to Preview the Message Template

The preview on the right side displays a visual representation of the template you've created.

&#x20;It showcases how the template will appear to users within the WhatsApp application, including any dynamic data that will be inserted when the template is used in actual conversations.&#x20;

Preview is essential for ensuring that the template is correctly formatted and that all variables are properly inserted, providing an accurate representation of how the message will look to users.
