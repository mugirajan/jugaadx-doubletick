## Send Template: Closed Chat Window

*<mark style="background-color:green;">**To perform this action via API, please visit**</mark>* [*<mark style="color:red;background-color:green;">**this link**</mark>*](https://doubletick.readme.io/reference/outgoing-messages-whatsapp-template)

Here's the process for sending a template to closed chat window:

* In the chat window, Click on <mark style="color:green;">**Send Template**</mark> button <img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FyQrs7aPj3hypWFvEt1js%2Fimage.png?alt=media&#x26;token=e0868970-87d6-469a-8f82-8201d005a131" alt="" data-size="line">

## Send Template: Open Chat Window

Here's the process for sending a template in an open chat window:

* In the chat window, click on the attachment icon <img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FAfXhPEl1kFGkoBEGjBd2%2Fimage.png?alt=media&#x26;token=dc6fc361-8396-4710-9c7c-42f1a945c7a9" alt="" data-size="line">
* From the popup menu, select the template icon&#x20;

<div align="left"><figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FLBsOwME4IHrrqmR3vNU1%2Fimage.png?alt=media&#x26;token=2aab5abb-4252-4da3-99c0-65faf209552c" alt="" width="262"><figcaption></figcaption></figure></div>

* Choose the desired template from the Template dialog box
* Proceed by clicking the green arrow button <img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FrVg66inzwNoBuAzLyivM%2Fimage.png?alt=media&#x26;token=eb667cc8-ef02-40fc-96bd-eb0f5b3049e1" alt="" data-size="line">
* Optionally, add any template variables if needed
* Finally, click "<mark style="color:green;">**Send Now**</mark>" to send the template to the recipient <img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FaNkUjFuy0o2tinaoxzMl%2Fimage.png?alt=media&#x26;token=93cc3afb-8b25-4fda-9cad-561f8c482fd9" alt="" data-size="line">
