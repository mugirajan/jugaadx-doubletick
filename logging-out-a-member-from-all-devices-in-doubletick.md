# Logging Out A Member From All Devices in DoubleTick

&#x20;**Step 1: Access the Logout Link**

Navigate to the logout link provided by Doubletick:

<https://doubletick.readme.io/reference/logout-team-member>

**Step 2: Enter the Login Number**

* On the logout page, locate the field for the **Login Number**.
* Enter the login number for the account you wish to log out of all devices.

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FGVSx255vaU7cdywtYsMZ%2FScreenshot%202024-11-19%20at%2020.06.05.png?alt=media&#x26;token=e0c5a2fa-8235-43d3-80cb-75dc3a8301fd" alt=""><figcaption></figcaption></figure>

**Step 3: Retrieve the API Key**

To authorise the logout process, you need the API Key:

1. Open the **DoubleTick** application.
2. On the left sidebar, hover over **Settings** and select **Developer's API**.
3. Copy the API key displayed under this section.

\\

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FuvYP330xYgGrdpaq1l85%2FUntitleddesign14-ezgif.com-video-to-gif-converter.gif?alt=media&#x26;token=085be83a-f134-4e2c-bbf7-c3d0e82cf1e3" alt=""><figcaption></figcaption></figure>

**Step 4: Provide the API Key**

1. Return to the [Logout Team Member](https://doubletick.readme.io/reference/logout-team-member) page.
2. Locate the **Header** field on the logout form.
3. Paste the API Key you retrieved in Step 3.

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FumTkOt5Uki9pkLWOQtFn%2FScreenshot%202024-11-19%20at%2020.29.59.png?alt=media&#x26;token=56340b2f-6841-4a3a-8771-66f57eb6fa62" alt=""><figcaption></figcaption></figure>

**Step 5: Complete the Process**

* Click the **Try It!** button on the logout page to submit the request.
* Wait for a confirmation message.
  * If successful, you will see a message that reads:\
    \&#xNAN;**`"success": true`**
  * This confirms the user has been logged out from all devices.
