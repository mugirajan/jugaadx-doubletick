# Send Template Messages

***

To configure the "Send Template Message" feature in settings, follow these steps:

* Navigate to settings and select "**Send Template Message**."
* Choose a media template from the dropdown menu&#x20;

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FEkO6kf4Od97YCfdtoxaL%2Fdrop%20down%20send%20message%20(2).gif?alt=media&#x26;token=e9eebb09-6e08-42aa-a4b2-00875503db67" alt=""><figcaption></figcaption></figure>

* Or click on "Create New Template" to design a new media template. Note that this new template can only be used once it's approved by Meta.

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FgYEUiVRTzeO8Mxef5hpn%2Fcreate%20new%20template%20send%20message.gif?alt=media&#x26;token=c3e30257-decf-4df7-9378-270a421f253d" alt=""><figcaption></figcaption></figure>

* After selecting the template, map customer attributes to the template as needed.
* Click on the "<mark style="color:green;">**Set as Default**</mark>" button to finalize the configuration

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FqFbI54986ij0Oya7xIMz%2FSet%20as%20default.gif?alt=media&#x26;token=3f60f2d1-3445-4abe-87ed-56daf2589ae9" alt=""><figcaption></figcaption></figure>

To utilize this feature, follow these steps on the **mobile application**:

1. Open a chat in the messaging interface.
2. Click on the "**Send Media**" button.
3. Select the multiple media files you wish to share.
4. Click on "**Upload**."
5. The selected media will be shared with the customer along with the template message configured previously. ðŸ‘‡
