# How to Send Dynamic content in a Single template  with DoubleTick

**Step 1: Create a New Template**

1. Navigate to the Templates section.
2. Click on New Template and select the appropriate utility category.
3. Fill in all the required fields to define your template.

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2F0lhh2tbHoT4yHOv4pgfi%2FUntitled%20design%20(2).gif?alt=media&#x26;token=c1f5e146-b7fe-4aae-a937-18f8d3650f8b" alt=""><figcaption></figcaption></figure>

**Step 2: Add Variables to Your Content**

1\) In the content section, create dynamic variables to personalise your message. For example: {{Text}}, {{Text2}}, and so on.

2\) These placeholders will later be replaced with actual text when sending the template.

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2F4wyTiaQlV4fz0e0VUvAy%2FUntitled%20design%20(4).gif?alt=media&#x26;token=02ab5e04-53cd-4589-9724-f55d6c96cc78" alt=""><figcaption></figcaption></figure>

**Step 3: Submit for Approval**

**1)** Once your template is ready, submit it for approval. The approval process ensures the template complies with the platform's standards and guidelines.

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FpggNIvBvmX8nGQoU4B5G%2FUntitled%20design%20(5).gif?alt=media&#x26;token=5bdf69a8-3be4-4dcf-bbbf-696136f6979a" alt=""><figcaption></figcaption></figure>

**Step 4: Add Actual Text for Variables After your template is approved:**

1. While sending the template, replace the variables with the actual text you want to send. oFor multiple variables, use a line break (\n) to separate them.
2. Click Send to deliver the customized message

<figure><img src="https://2303112206-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F53n17VnOICC1LtDqlENV%2Fuploads%2FeBqmnkb3soBcNxBsvNNT%2FUntitled%20design%20(7).gif?alt=media&#x26;token=ebeab82c-1e39-40a1-a79f-6a5a11ec957a" alt=""><figcaption></figcaption></figure>

**Step 5: View by the End User** \
\
The recipient will see the actual text content with proper formatting and line breaks, making the message clear and personalized\
