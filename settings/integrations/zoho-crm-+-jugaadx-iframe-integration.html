Zoho CRM + JugaadX IFrame Integration

With the Zoho IFrame integration, you can use JugaadX directly inside Zoho CRM. That means:

  • No switching between tabs

  • No copying phone numbers

  • No jumping between tools

  • WhatsApp messaging becomes super easy and fast

This guide explains everything in simple language, even if you’ve never set up an integration before.

💡What Does This Integration Do?

Think of the Zoho → JugaadX integration like placing a small “window” inside Zoho CRM.

Through this window:

➡️ You can use JugaadX ➡️ You can message your leads ➡️ You can view conversations ➡️ You can reply instantly

…all without leaving Zoho CRM.

So instead of switching between Zoho and JugaadX, everything comes together in one place.

This “window” is called an IFrame Widget.

📌 What You Need Before You Start

  • A Zoho CRM account

  • Admin access (only admin can add widgets)

  • A widget file from JugaadX (your Account Manager/Support Team will give this file )

  • Correct phone numbers saved in Zoho CRM Leads/Contacts

  • Your JugaadX user phone number must match your Zoho CRM user number

Don’t worry — everything is explained below.

🛠 How to Set Up the Zoho IFrame Integration (Step-by-Step)

Follow these steps exactly — no technical skills required.

Step 1: Open Zoho CRM Setup

  1. Log in to Zoho CRM.

  2. Look at the top right corner → click the Setup/Settings icon (usually a wrench or screwdriver symbol).

This opens Zoho’s main settings menu.

Step 2: Search for “Widgets”

  1. In the Setup search bar, type Widgets.

  2. Click on Widgets when it appears in the list.

Widgets are small tools you add inside Zoho — JugaadX will become one of them.

Step 3: Create a New Widget

Now you will create the “window” through which JugaadX will appear.

  1. Click Create New Widget.

  2. A form will open.

  3. Give your widget a simple name like JugaadX Widget.

  4. Upload the widget file you received from JugaadX Support. (This file is very important — it tells Zoho how to load JugaadX inside the CRM.)

  5. Save it.

That’s it — your widget is now created.

Step 4: Add the Widget to Your Lead View

Now you must attach this widget to the Lead or Contact section.

  1. Open any Lead inside Zoho CRM.

  2. Scroll all the way down to the bottom of the page.

  3. Click Add Related List.

  4. You will see a list of items — choose the JugaadX Widget you just created.

  5. Save it.

From now on, this widget will appear for every lead automatically.

Step 5: Make Sure the Lead Has a Phone Number

The JugaadX widget loads only when Zoho has a valid phone number.

Important notes:

  • Enter the number with a country code Example: +91 9876543210

  • If the phone number is missing, Zoho will ask you to add it

  • Once added, the widget will refresh and load JugaadX

This is because JugaadX needs the phone number to show your chat history.

⚠️ VERY IMPORTANT — User Number Matching

For the widget to work properly:

  • Your JugaadX user phone number (login number) must match

  • Your Zoho CRM user phone number

If they do not match:

  • The widget may load but messaging won’t work

  • You won’t see chat history

  • Some buttons may not appear

This is a common mistake, so please double-check this.

If your numbers don’t match, ask your Zoho admin to update your Zoho CRM phone number.

🎉 You’re Done!

Your Zoho CRM now has JugaadX built inside it.

You can now:

  • Open any Lead/Contact

  • See the JugaadX panel instantly

  • Send WhatsApp messages directly

  • View previous conversations

  • Reply to chats without opening multiple tabs

Your daily workflow becomes 10× faster and much easier.
