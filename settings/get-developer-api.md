# Get Developer API

***

Integrate third-party apps with DoubleTick using our Developer API. Access API documentation and API keys for seamless integration and enhanced workflow efficiency.

Integrating your DoubleTick account with third-party software can significantly enhance your workflow efficiency. Our Developer API page offers the necessary tools, including API documentation and an API key, for seamless integration.

#### **Accessing the Developer API**

* Go to the **"Settings"** section on your dashboard
* Click on **"Developer API"** to access the integration tools

***

#### **Utilizing the API Key**

* The API key provided is your authorization to access the DoubleTick API
* Remember, the API key grants access to your entire account, so share it cautiously

#### **Accessing API Documentation**

* Click on the provided link to access comprehensive API documentation
* The API documentation includes DoubleTick Public APIs and Webhooks that can be integrated with third-party applications. You have access to various modules such as **templates, broadcasts, wallets, teams, roles**, and more.
* Each module offers specific functionalities that can be integrated with third-party applications

***

#### **Collaboration with Developers**

Share the API key and documentation with your developers for system-based integrations like **SAP** or **ERP**. This ensures they understand the integration process and implement it effectively.

***

By following these simple steps and leveraging our **Developer API page**, you can seamlessly integrate your DoubleTick account with third-party software, unlocking new efficiencies and enhancing productivity in your workflow.
